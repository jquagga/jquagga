# Hello! My name is Justin

## Patcher, Tinkerer and occasional Coder

This is a playground for various projects I've worked on are am working on. I have some Python AI code which transcribes audio to text using Whisper.  I also am experimenting with the NOAA weather API and building a webapp in Svelte that shows the local weather in a more mobile friendly format.  I also have a few docker stacks related to software define radio projects. I enjoy learning from great things others have built and put them together in new ways to hopefully do new things!

- 🌍  I'm on Earth in the Sol system.
- 🧠  I'm learning Python, Svelte, OpenStreetMap and everything I bump into along the way.
